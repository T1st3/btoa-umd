btoa-umd

btoa-umd is a Javascript UMD module for btoa().

This module is written using the UMD definition of modules. It enforces the returnExports.js pattern, which “defines a module that works in Node, AMD and browser globals”.

So you can use this module in the following contexts:

  • Node.js
  • Browser globals
  • AMD (with Require.js or any other AMD loader)

Demo (using AMD modules)

What we'll do here is just a simple wrapper function around the AMD module, and a simple use case.

The demo will use the wrapper to encode the string Hello world, which is the decoded version of SGVsbG8gd29ybGQ=. 

  
$(document).ready(function () {
  $('#in').on('click', function () {
    var btoa = function (b) {
      var umd = new Btoa();
      return umd.handle(a).b;
    };
    var res = btoa('Hello world');
    $('#out').html(res);
  });
});
